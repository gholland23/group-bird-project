WE LIKE BIRDS

As a group, we decided to branch out the box for this project. The reason being is that the majority of the world right now is sad. The COVID-19 pandemic has not only taken over the world, but it also invaded Kaggle. While scanning the website for topics, it seemed like every freaking dataset was about COVID and its stats. Then, we came upon a dataset about birds...and well no pun intended, but our imaginations took flight.

QUESTIONS WE WANTED TO ANSWER!

  • "Where are these birds located?"
  • "Where do they migrate to?"
  • "Is it safe to say birds migrate to the equator?"

    • HOW DID WE DO IT?

  • Our original dataset only contained the names of 150 birds and pictures that associated each bird. In order to answer these questions we had to look up and generate coordinates for each bird and create a true database. Out of the 150 birds we chose 30 to look up origin and migration coordinates of. After this data was collected, Nam used Postgres to create that database.
  • The second step was to use leaflet to map out the bird's location (origin markers).
  • Then using the destination markers we would use leaflet to create lines demonstrating the migration patterns of our birds.
  • Lastly, we plotted markers using the destination locations and plotted a shape around the equator with appropriate measurements in hopes of ansewring our question.
